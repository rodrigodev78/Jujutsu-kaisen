# *Jujutsu Kaisen*

Um jogo de luta inspirado em Jujutsu Kaisen, construído com *HTML, CSS e JavaScript puro*.  
Permite que dois jogadores escolham personagens, batalhem e vejam animações de ataque em tempo real.

## *Funcionalidades*

- *Seleção de personagem* para dois jogadores  
- *Barra de vida* dinâmica para cada personagem  
- *Animações CSS* de ataque  
- *Log de dano* em tempo real  
- Tela de *vitória/derrota*  
- Totalmente *responsivo*, adaptando-se a diferentes tamanhos de tela

## *Tecnologias*

- *HTML5* – Estrutura do jogo  
- *CSS3* – Estilização, animações e responsividade  
- *JavaScript (ES6)* – Lógica de jogo, OOP, controle de personagens e ataques

## *Como jogar*

1. Abra o arquivo index.html no navegador  
2. Selecione o *personagem do jogador 1* e do *jogador 2*  
3. Clique em *Começar* para iniciar a batalha  
4. Clique em *Atacar* para realizar golpes alternados  
5. A batalha termina quando a barra de vida de um dos personagens chega a *0*  
6. Clique em *Recomeçar* para jogar novamente
